# Security Policy

## Supported Versions

The following versions of AiAyN are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to Chaitanya Nirhali at chaitanyanirhali@gmail.com. All security vulnerabilities will be promptly addressed.

Please include the following information in your report:

- Type of issue (e.g., XSS, SQL injection, etc.)
- Location of the vulnerability
- Potential impact
- Steps to reproduce

We appreciate your help in keeping this project secure!
